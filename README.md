# 🎨 DrawTogether v2

Интерактивное веб-приложение для совместного рисования в реальном времени, разработанное специально для образовательного процесса. Позволяет учителю проводить игровые раунды, задавать темы, а ученикам — синхронно рисовать на общем холсте и получать мгновенную оценку.

## ✨ Особенности

- 🧑‍🏫 **Две роли**: Учитель (администратор комнаты) и Ученик (участник)
- 🔗 **Быстрый вход**: Уникальный 6-символьный код комнаты или прямая ссылка
- 🎛️ **Гибкие настройки**: Количество раундов, время, размер/цвет холста, доступность ластика, показ слова ученикам
- 🖌️ **Набор инструментов**: Кисть, ластик, линия, прямоугольник, эллипс, аэрограф
- ⏱️ **Таймер раундов**: Визуальный обратный отсчёт с пульсирующим предупреждением
- ⭐ **Система оценки**: Учитель выставляет оценку (1–5⭐ или без оценки) после каждого раунда
- 🖱️ **Курсоры в реальном времени**: Видно движение и имена других участников
- 💾 **Экспорт**: Сохранение итогового рисунка в PNG
- 📱 **Адаптивный UI**: Плавные анимации, современный дизайн, поддержка мобильных устройств

## 🛠 Технологический стек

| Слой        | Технологии                          |

## 📦 Установка и запуск

### 1. Клонирование и установка зависимостей

```bash
git clone <your-repo-url>
cd DrawTogether
npm install
```

### 2. Запуск сервера

```bash
# Для разработки (с автоматическим перезапуском через nodemon)
npm run dev

# Для продакшена
npm start
```

Приложение будет доступно по адресу: `http://localhost:3000`  
*(Порт можно изменить через переменную окружения `PORT`)*

## 🎮 Как использовать

### 👩‍🏫 Для учителя

1. Нажмите **«Создать комнату»**, настройте параметры холста и игры.
2. Скопируйте код комнаты или ссылку и отправьте ученикам.
3. Дождитесь подключения учеников и нажмите **«Начать игру»**.
4. Введите слово для раунда и нажмите **«Начать раунд»**.
5. По окончании времени оцените рисунок (⭐ 1–5 или пропустить).
6. Переходите к следующему раунду или завершите игру для просмотра итоговой таблицы.

### 🎒 Для ученика

1. Введите 6-символьный код комнаты и своё имя.
2. Дождитесь старта раунда.
3. Рисуйте на общем холсте, используя выбранные инструменты.
4. По окончании раунда дождитесь оценки учителя.
5. В конце игры просмотрите общую таблицу результатов.

## 🛡️ Безопасность и ограничения

- Комнаты автоматически удаляются из памяти через **4 часа** неактивности.
- История рисования ограничена `20 000` событиями на комнату для предотвращения переполнения памяти.
- Коды комнат генерируются из набора `ABCDEFGHJKLMNPQRSTUVWXYZ23456789` (исключены похожие символы).
